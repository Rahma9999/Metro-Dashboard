## Cairo Metro Dashboard
A web application for managing metro stations and subscriptions, built with React. It allows administrators to view stations, track tickets, and manage subscriptions.

## Features
- login system
- Station management
- ticket management
- Subscription system
- Responsive UI
- Line and station analysis 

## Tech Stack
- React.js
- JavaScript (ES6+)
- Bootstrap / CSS
- Axios
